# Explore Tourist Places

A simple, responsive website showcasing popular tourist destinations in Mumbai, India. This project highlights iconic landmarks and attractions, providing information and beautiful visuals for travelers.

## Features

- **Responsive Design**: Optimized for desktop and mobile devices
- **Hero Section**: Eye-catching introduction with background image
- **Tourist Places Gallery**: Cards displaying top places to visit in Mumbai
- **Interactive Form**: Simple form for users to share their favorite places
- **Clean Navigation**: Sticky navigation bar for easy browsing

## Tourist Places Featured

1. **Gateway of India** - Historic monument and iconic landmark
2. **Juhu Beach** - Popular beach with street food and sunsets
3. **Marine Drive** - Famous promenade known as the "Queen's Necklace"
4. **Trimbakeshwar Jyotirling** - Ancient Hindu temple
5. **Shree Siddhivinayak Temple** - Popular Ganesh temple in Mumbai

## Technologies Used

- HTML5
- CSS3 (with responsive design using media queries)
- Font Awesome (if icons are used, though not visible in code)

## How to Run

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. Navigate through the sections using the menu or scroll
